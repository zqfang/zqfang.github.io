My bioinfomatcis workflows using snakemake.

Why snakemake?  

Good:
- Easy to use and update
- Reusable workflows
- Scalable
- Support conda env
- Support slurm, qsub ...
- python syntax

Drawback: 
- Hard to debug workflows.
